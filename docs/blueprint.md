# **App Name**: System Architect Portfolio

## Core Features:

- Project Index: Display a scrollable index of system architecture projects with key details.
- Case Study Viewer: Provide an in-depth view of individual case studies with project details, diagrams, and outcomes.
- Article Reader: Showcase a collection of articles and notes.
- Interactive System Diagram Generator: AI tool that will generate an interactive diagram of a given software system based on a natural language description.
- Contact Form: Implement a contact form for users to submit inquiries and messages.

## Style Guidelines:

- Primary color: Deep Indigo (#3F51B5) to convey trust and expertise.
- Background color: Dark gray (#263238) for a professional, modern feel.
- Accent color: Teal (#009688) to highlight interactive elements.
- Body and headline font: 'Inter' (sans-serif) for a clean, readable interface.
- Code font: 'Source Code Pro' for displaying code snippets.
- Use minimalist line icons from Lucide to represent system components and actions.
- Subtle animations for transitions and interactions to enhance the user experience.
